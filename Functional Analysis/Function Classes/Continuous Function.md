# Continuous Function
Let $I$ be an interval. We define the set of [[Continuous Function]]s to be of the form:
$$\large C(I) = \{ f : I \to \mathbb{C} | f \text{ is continuous} \} $$
We have that $C=C(\mathbb{R})$, the set of continuous functions defined on $\mathbb{R}$. a
