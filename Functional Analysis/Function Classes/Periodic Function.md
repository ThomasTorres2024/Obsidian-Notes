# Periodic Functions 
Let $I$ be an interval, finite or infinite. The class of periodic functions here is the following:
$$\large \{ f: \mathbb{R} \longrightarrow \mathbb{C} | f \text{ is periodic with period }p>0  \}$$
Which is to say that, $f(x+p)=f(x) : \forall x \in \mathbb{R}$. We can determine the values of $f$ anywhere based on any interval of size $p$. The smallest period of a periodic function is called the "fundamental period." For example, $\large e^{2\pi i}$ has a period of $2 \pi$. 

If the following series converges, then it has period of $2\pi$:
$$\large \sum_{n=-\infty}^\infty c_{n} e^{inx}$$
