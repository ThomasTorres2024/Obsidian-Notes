# Bessel's Inequality 
Let Let $(\mathcal{H},\langle , \rangle )$ be a [[Hilbert Space]]. Given the [[Orthonormal]] set in $\mathcal{H}$, $\{ \vec{u_{1}} ,\vec{u_{2}}, \dots, \vec{u_{n}}  \}$, then for any $\vec{x} \in \mathcal{H}$:
$$\large \sum_{i=1}^{n} |\langle \vec{x}, \vec{u_{i}} \rangle|^{2} \leq \|x\|^2$$
It turns out that if the orthonormal set spans $\mathcal{H}$, then we obtain the following equality:
$$\large \sum_{i=1}^{n} |\langle \vec{x}, \vec{u_{i}} \rangle|^{2} = \|x\|^2$$
Which is actually [[Parseval's Theorem]].
