# Morphism
A [[Morphism]] is a [[Function]] that maps elements from [[Object (Category Theory)]] $A$ to object $B$. Morphisms can be composed together.  

A [[Morphism]] can also map back to itself, this is known as an [[Endomorphism]]. 

In the context of Abstract Algebra we tend to only consider morphisms between Algebraic structures such as [[Group]]s, [[Field]]s, and [[Ring]]s.
