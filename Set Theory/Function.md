# Function 
A [[Function]] is a [[Relation]] between $2$ [[Set]]s, $U,V$ where $f:U\to V$ and $\text{dom}(f)=U \quad \text{codomain}(f)=V$ and for $a,b \in U$ and $x,y \in V$ we have that $f(a) \in V$, and that $(a,x),(a,y) \in V \implies x=y$.  

---
# Function Composition
If we have the functions $f,g,h$ function composition actually turns out to be associative and thus we can express that the following is true:

$$h \circ (g \circ f) = (h\circ g) \circ f =h \circ g \circ f$$
