# Equivalence Relation 
An [[Equivalence Relation]] is a [[Relation]] which satisfies the $3$ criteria of reflexivity, symmetry, and transitivity. Let $R$ be a relation.

1. Symmetry $xRy \implies yRx$
2. Reflexivity $xRx$
3. Transitivity $xRy, yRz \implies xRz$
