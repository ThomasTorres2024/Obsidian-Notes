# Relation
A relation $R$ on a set $U \to V$ is a subset of the [[Cartesian Product]] between $2$ [[Set]]s such that for each $(x,y )\in R$ where $x \in U, y  \in V$ we have that $(x,y)$ satisfies some binary condition. For instance if $U=V=\mathbb{Z}$ and $R$ is the condition is even, then all $(x,y) \in R$ consist of $x$ and $y$ being even integers.
