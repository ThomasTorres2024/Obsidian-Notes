# Definition of Invariant Subspaces 

We define an "operator" as a linear transformation $\mathcal{L}$  where $\mathcal{L} : V \rightarrow V$ for vector space $V$. 

In terms of Group/Category theory, this can be thought of as an [[Endomorphism]].
