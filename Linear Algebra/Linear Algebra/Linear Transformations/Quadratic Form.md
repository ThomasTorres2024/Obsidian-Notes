# Quadratic Form Overview
For some $A \in \mathbb{R}^{n \times n}$ the [[Quadratic Form]] of $A$, for some $x,b \in \mathbb{R}^n$ is:
$$f(x)=\frac{1}{2}x^TAx-b^Tx$$
The [[Quadratic Form]] is a Bilinear Form ([[Bilinear Forms]]). If we take the [[Gradient]] of this expression, we obtain the following [[system of equations]]:
$$Ax=b$$
