# Kronecker Delta
A function for integers $i,j$ where in some more meaningful context (in the context of linear algebra typically [[orthonormal]] vectors from sets ) satisfy the following:
$$ \langle v_{i},v_{j}  \rangle = \delta_{ij}= \begin{cases}
1 & \text{if } i =j\\ 0 & \text{if} i \neq j \\
\end{cases}$$
