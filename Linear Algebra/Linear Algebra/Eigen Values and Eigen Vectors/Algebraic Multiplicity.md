# Algebraic Multiplicity Definition
Given some $A \in \mathbb{F}^{n \times n}$ with the characteristic equation $p(x)$ with [[Eigen Value]]s as roots, the [[Algebraic Multiplicity]] of an [[Eigen Value]] is the number of times the [[Eigen Value]] solves the [[Characteristic Polynomial]].
