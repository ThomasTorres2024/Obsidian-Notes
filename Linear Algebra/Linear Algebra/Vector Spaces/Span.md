# Span Definition
The [[Span]] of a set of [[Vector]]s is the set of all of their [[Linear Combination]]s. Let the set of vectors $S$ be the following over the field $\mathbb{F}$:
$$S=\{ v_{1},v_{2},\cdots, v_{n} \}$$
Then the [[Span]] of $S$ is the following:
$$\text{span}(S)= \sum_{i=1}^n  \alpha_{i} v_{i} : \alpha_{i} \in \mathbb{F} $$
