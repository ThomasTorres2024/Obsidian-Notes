# Functional Language
A [[Functional Language]] is a [[Fundamentals of Programming Languages/Languages Overview/Programming Language]] which goes about solving problems by continual recursive calls on data and using mathematical functions. Typically, unless for [[IO]], we do not tend to modify the underlying data in our program, which is what distinguishes this modality of programming from imperative programming.
