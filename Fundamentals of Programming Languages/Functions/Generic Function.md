# Generic Function 
In a [[Functional Language]], we say that a [[Generic Function]] is a kind of function which changes it behavior depending on the kind of function it receives as an input. 

For instance, consider the following:
```asteroid 
function c with (f,v) do 
	return f(v).
end
```
Then we compute the value of $f$ at $v$.
