# Operation (Category Theory)
If we consider an [[Endofunctor]] $f$ that maps in [[Category]] $A\to A$, there are "operations" on the category which take 2 elements in $A$ and return another value. 

Some of these common properties include closure, and thus form a [[Binary Operation]] since a necessary condition is closure. In category theory this is called "Totality". There is also associativity.
