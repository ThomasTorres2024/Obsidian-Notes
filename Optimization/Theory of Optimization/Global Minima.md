# # Global Minima 
For a function $f : \mathbb{R}^n \mapsto \mathbb{R}$, $f$ obtains a [[Global Maxima]] at point $x^*$, where $x^*$ satisfies that $f(x^*) \leq f(x) : \forall x \in D, x  \neq x^*$. It may be possible that a function has more than one minima, so this is not a unique value necessarily.
