# Global Maxima 
For a function $f : \mathbb{R}^n \mapsto \mathbb{R}$, $f$ obtains a [[Global Maxima]] at point $x^*$, where $x^*$ satisfies that $f(x^*) \geq f(x) : \forall x \in D, x  \neq x^*$. It may be possible that a function has more than one maxima, so this is not a unique value necessarily.
