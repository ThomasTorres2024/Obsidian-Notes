# Position Vector Field 
The [[Position Vector Field]] is a [[Vector Field]] defined for $\mathbb{R}^n$ by $\vec{v}$ such that the $i$th component of $\vec{v}$ corresponds to the position along the axis of $x_i$. More concretely for $\mathbb{R},\mathbb{R}^2,\mathbb{R}^3$:
$$\langle x\rangle, \langle x,y \rangle, \langle x,y,z\rangle$$
Generally we can describe it by:
$$\vec{r} : \mathbb{R}^n \to \mathbb{R}^n, \vec{r}=\langle x_1, x_2, \dots x_n \rangle$$
