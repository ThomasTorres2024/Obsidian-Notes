# Del Operator
The [[Del]] operator is expressed, for some [[Vector Field]] $\vec{F} : \mathbb{R}^n \mapsto \mathbb{R}^n$ where $\vec{F}=\langle X_1,X_2,\dots, X_n \rangle$ is the vector field definition, can be described by the following:
$$\nabla \cdot \vec{F}=\sum_{i=1}^n\frac{\partial X_i}{\partial x_i}=\frac{\partial X_1}{\partial x_1}+\frac{\partial X_2}{\partial x_2}+\dots \frac{\partial X_n}{\partial x_n}$$

---
# Del Operator identities:
* $\nabla (f+g)=\nabla f + \nabla g$
* $\nabla(cf)=c \nabla f$
* $\nabla(fg)=(\nabla f) g + f (\nabla g)$ 
* $\nabla \left( \frac{f}{g} \right) = \frac{(\nabla f)g-f(\nabla g)}{g^2}$
