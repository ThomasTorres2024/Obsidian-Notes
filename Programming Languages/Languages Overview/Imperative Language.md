# Imperative Language
[[Imperative Language]]s are [[Programming language]]s which supports variables being modified and reassigned. This type of language is best suited to work with iteration and continual reassignment. 

Most popular programming languages in the modern day are [[Imperative Language]]s, like [[Python]],[[Dart]], [[Java Script]], [[C]], [[C++]], etc.
